## Installation

1. Download from Chrome Web Store (coming soon)
2. Or install locally:
   ```bash
   # Clone the repository
   git clone https://github.com/nerdynikhil/monkey-speak.git
   
   # Load in Chrome
   1. Open Chrome and go to chrome://extensions/
   2. Enable "Developer mode"
   3. Click "Load unpacked"
   4. Select the monkey-speak folder
   ```

## Usage

1. Visit [Monkeytype](https://monkeytype.com)
2. Click the MonkeySpeak extension icon
3. Toggle speech on/off using the button
4. Start typing and hear words spoken in real-time
